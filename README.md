# GlassedProject
# 一杯子商家端

应用说明: 该应用是管理"一杯子"的应用, 是对"一杯子"中商品的管理、订单的处理以及店铺的管理的应用。

主要功能: 
      1、商品管理-对商品进行上架下架、修改、增添等功能； 
      2、订单管理-对订单进行操作、订单流程的处理； 
      3、二维码管理-通过扫描订单二维码获取订单信息； 
      4、店铺管理-对店铺进行修改、增添等功能； 
      5、收藏管理-查看用户收藏的统计； 
      6、评价管理-查看用户所提交的评价数据； 
      7、消息管理-查看订单消息和系统消息。
